About This App

This application allows users to upload fabric images for classification. The system predicts whether the fabric is Corduroy, Denim, or Linen. You can also view the classification report for previous predictions.

How It Works

  1. Go to the "Upload Image for Prediction" page.
  2. Select an image of the fabric (.png, .jpg, or .jpeg format).
  3. The system will analyze the image and return the predicted label.
  4. You can also view the classification report for detailed analysis.
