About This App

This application allows users to upload fabric images for classification. The system predicts whether the fabric is Corduroy,or Denim.

How It Works

  1. Go to the "Upload Image for Prediction" page.
  2. Select an image of the fabric (.png, .jpg, or .jpeg format).
  3. The system will analyze the image and return the predicted label.
